Data Sampling Techniques
This project compares three important sampling methods used in statistics and data science:
Random Sampling
Systematic Sampling
Stratified Sampling
The purpose of this project is to understand how different sampling techniques affect the accuracy, fairness, and quality of collected data. The same dataset is used for all three methods so that the comparison becomes easier and more understandable.


Tools Used
Excel
Python


Day 1


Problem Statement
The goal of this project is to study different sampling methods and compare their performance on the same dataset. Sampling is important because collecting data from an entire population is difficult and time-consuming.

Random Sampling

Systematic Sampling

Day 5


Stratified Sampling
